# 🇬 Проект: Конвертер баллов IELTS → уровень английского

**Тип проекта:** WEB-тестирование  
**Объект тестирования:** Форма ввода баллов IELTS с выводом уровня CEFR  
**Ссылка на приложение:** [brainy-qa.github.io/ielts-score](https://brainy-qa.github.io/ielts-score/)  
**Дата:** [15.03.2026]

---

## Описание задачи

Веб-приложение принимает балл IELTS (0.0–9.0) и отображает соответствующий уровень владения языком по шкале CEFR (A1, A2, B1, B2, C1) с визуальным отображением.

**Требования к вводу:**
- Поле обязательно для заполнения
- Разрешены только цифры и точка 
- Запрещён ввод букв, спецсимволов, нескольких точек
